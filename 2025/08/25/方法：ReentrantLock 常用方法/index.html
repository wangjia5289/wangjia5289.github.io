Last updated on 2025-08-28T18:21:31+08:00

构造方法

public ReentrantLock()

public ReentrantLock(boolean fair)

实例方法

加锁方法

public void lock()

[!NOTE] 注意事项

  1. 线程的中断标志位仍然会被设置为 true

public void lockInterruptibly()

public boolean tryLock()

public boolean tryLock(long timeout, TimeUnit unit)

解锁方法

public void unlock()

[!NOTE] 注意事项

  1. 必须由持有锁的线程进行调用，否则会抛 IllegalMonitorStateException
  2. 始终在 finally 块中调用 unlock()，这样才能保证无论加锁代码是否抛出异常，锁都能被正确释放。
