方法：FileChannel 常用方法

Last updated on 2025-08-27T21:38:24+08:00

构造方法

FileChannel 可以通过 FileInputStream、FileOutputStream、RandomAccessFile 的 getChannel() 方法获取：

  1. 使用 FileInputStream 获取的是读通道
  2. 使用 FileOutputStream 获取的是写通道
  3. 使用 RandomAccessFile 获取的是读写通道

[!NOTE] 注意事项

  1. 他们的 API 方法列表是完全一致的，区别在于他们只能调用符合权限的方法，否则会抛异常。
    1. 例如通过 FileInputStream 获取的读通道如果调用 write()，会抛出异常
    2. 例如通过 FileOutputStream 获取的写通道如果调用 read()，同样会抛异常
    3. 而使用 RandomAccessFile 获取的读写通道则没有这些限制，读写操作都支持

实例方法

读方法

int read(ByteBuffer)

[!NOTE] 注意事项

  1. 一定要牢记这两个流程：
    1. 程序读数据：硬件（磁盘/网卡） ➔ 内核空间 ➔ Channel ➔ Buffer
    2. 程序写数据：Buffer ➔ Channel ➔ 内核空间 ➔ 硬件（磁盘/网卡）
  2. 如果你发现文件内容莫名消失，原因可能是你创建的 FileOutputStream 没有开启追加模式，只需在创建时设置为追加模式即可：

int read(ByteBuffer, long)
