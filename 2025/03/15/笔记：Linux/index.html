总结

步骤 操作
更新系统 sudo apt update && sudo apt upgrade -y
更换软件源（国内可选） 修改 /etc/apt/sources.list
安装常用工具 vim, curl, git, net-tools
修改主机名 hostnamectl set-hostname myserver
配置 SSH 修改 sshd_config，更改端口、禁用 root 登录
创建新用户 adduser myuser && usermod -aG sudo myuser
设置防火墙 ufw allow 22/tcp，然后 ufw enable
配置时区 timedatectl set-timezone Asia/Shanghai
配置 Swap fallocate -l 2G /swapfile && mkswap /swapfile
安装 Docker apt install docker.io && usermod -aG docker myuser
安装开发环境 Java / Node.js / Python（按需安装）

这样你的 Ubuntu 就基本上配置好了！后续可以根据自己的需求进一步优化 🚀
