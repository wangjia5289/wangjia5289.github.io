笔记：IP

Last updated on 2025-09-05T21:49:46+08:00

1. IP 概述

IP（Internet Protocol）是互联网协议地址，也称 IP 地址，是分配给上网设备的数字标签

2. IP 分类

  1. IPv4
  2. IPv6

3. IPv4

3.1. IPv4 概述

IPv4（Internet Protocol version 4）是互联网通信协议第四版，采用 32 位地址长度，分成 4 组，每组 8 bit

[!NOTE] 注意事项

  1. 常见的 192.168.0.0/16 这类 “斜杠+数字” 写法叫做 CIDR 表示法。其中的 /16 表示前 16 位固定，其他位数可变化。因此，192.168.0.0/16 实际上表示的地址范围是 192.168.0.0 ~ 192.168.255.255。

3.2. 特殊 IPv4 地址

127.0.0.1（也可写作 localhost）是回环地址，又称本地回环 IP 或 本机地址。它的作用是指向当前这台主机本身，无论何时都会回到本机，而不会访问网络中的其他设备。

举个例子：如果你的电脑在局域网中获得的地址是 192.168.1.100，那么这表示路由器分配给你的一般局域网 IP。而 127.0.0.1 永远只指向你自己的电脑，两者含义完全不同。

[!NOTE] 注意事项

  1. 除此之外，还有：
    1. 0.0.0.0
      1. 表示未指定的地址，没有固定身份
      2. 在系统中绑定/监听 0.0.0.0，表示监听本机所有网卡的地址
    2. 224.0.0.0 - 239.255.255.255
      1. 组播地址，用于一对多通信
      2. 其中 224.0.0.0 - 224.0.0.255 为预留的组播地址
    3. 240.0.0.0 - 255.255.255.244
      1. 保留地址，目前未用于公网
    4. 255.255.255.255
      1. 广播地址，用于向本地网络中所有主机发送消息

3.3. 缓解 IPv4 不够问题

为缓解 IPv4 地址数量不足的问题，互联网将地址划分为公网地址（用于万维网）和私有地址（用于局域网）。其中，私有地址的一段是 192.168.0.0 ~ 192.168.255.255，常见的 192.168 开头的 IP 就属于这一范围，从而有效节省了有限的 IPv4 资源。

4. IPv6

IPv6（Internet Protocol version 6）是互联网通信协议第六版，采用 128 位地址长度，分成 8 组，每组 16 bit
